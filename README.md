# Animated Face Using HTML & CSS

A simple and playful **animated face** built entirely with HTML and CSS.  
The face smoothly moves up and down using CSS keyframe animations to create a lively effect â€” no JavaScript needed.

---

##  Features

- Fully responsive and lightweight  
- Smooth animation using `@keyframes`  
- Creative use of CSS positioning and `border-radius` for facial features  
- Pure HTML & CSS, no external libraries

---

##  Technologies Used

- **HTML5**  
- **CSS3 (Animations, Keyframes, Positioning)**  

---

##  How to Run

1. **Clone this repository**
   ```bash
   git clone https://github.com/mirajamous/animated-face.git
   ```
2. Open the folder and launch `index.html` in your browser.
